K-nearest neighbors

Click on the canvas to create new data points and see its class according to KNN algorithm

  • This visualization tool shows how the KNN algorithm classifies a data point.

    When you add a new data point:

    1. The algorithm checks the distance of this new point from all the other points.
    2. The distance from the sample and the class of the sample are added to an ordered collection.
    3. The ordered collection of distances and classes are sorted in ascending order of the distances.
    4. The mode of first K labels becomes class of the the new data point.
